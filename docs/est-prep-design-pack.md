# EST Prep Design Pack

## Module Title
EST Prep: Decode, Recall, Respond

## Purpose
Prepare students for the upcoming Careers and Employability EST by combining:
- targeted revision of the assessed content
- explicit teaching of command verbs
- repeated decoding of EST-style questions
- structured answer building based on likely marking expectations

This module should feel like a high-stakes training simulation inside Career Empire rather than a worksheet bank.

## Core Learning Goals
Students should be able to:
- identify the command verb in an EST question
- identify the topic being assessed
- identify the context or glossary term that narrows the question
- choose an answer structure appropriate to the command verb and mark allocation
- recall the relevant curriculum knowledge from the assessed revision topics
- construct concise, mark-worthy EST responses

## Design Principles
- Keep the game pressure high but supportive
- Reward precision, not just speed
- Use real EST-style wording often
- Make marks visible and meaningful
- Save teacher-visible evidence of student responses
- Let students improve answers after feedback

## Gameplay Loop
1. Receive an EST-style question
2. Decode it using VTCS
3. Match the relevant glossary concept or topic
4. Choose the best answer structure
5. Write a short answer
6. Compare against mark-style feedback
7. Earn rewards and improve readiness

## VTCS Engine
The module should repeatedly train:
- V: Verb
- T: Topic
- C: Context
- S: Structure

Students should not move straight to answering until they decode the question correctly or receive scaffolding.

## Recommended Module Structure
### Chapter 1: Decode The Question
Purpose:
- teach students how to unpack EST questions quickly and accurately

Gameplay:
- highlight the command verb
- drag or tap to identify topic and context
- choose the correct answer structure

Evidence:
- decoded-question record

Key content:
- identify
- outline
- describe
- explain
- discuss
- compare
- analyse

### Chapter 2: Glossary Lock-In
Purpose:
- connect glossary terms to question contexts

Gameplay:
- match term to definition
- distinguish similar terms
- identify which term is the real context in a question

Evidence:
- glossary match accuracy

Examples:
- career planning
- growth industry
- initiative
- megatrend
- personal risk
- professional risk

### Chapter 3: Revision Arena
Purpose:
- build content recall for the assessed topics in the revision sheet

Gameplay:
- timed recall rounds
- scenario-to-topic matching
- "choose the strongest answer point" tasks

Assessed topics from the current revision sheet include:
- communication skills
- time management skills
- unexpected life events and career planning
- strategies to manage personal finance
- labour market information
- growth industries
- initiative
- cover letters
- selection criteria techniques
- megatrends and future of work

### Chapter 4: Build The Answer
Purpose:
- move from decoding to actual response construction

Gameplay:
- students assemble answer parts in correct order
- then write their own answer

Structure training examples:
- Identify / State: one point
- Outline: what it is + key feature
- Describe: features or characteristics
- Explain: point + because/how + result
- Discuss: point + explanation + example + result
- Compare: similarities + differences
- Analyse: evidence + meaning + conclusion

### Chapter 5: Marking Key Challenge
Purpose:
- train students to think like the marker

Gameplay:
- compare sample answers
- decide which would earn more marks
- identify what is missing
- improve a weak answer

Evidence:
- answer improvement artifact

- how many ideas are needed
- how to structure an answer for marks

That is the most powerful bridge between curriculum knowledge and actual EST success.
