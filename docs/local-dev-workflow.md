# Local Dev Workflow

Run a static local server from the project root:

```bash
npm run dev
```

Then open:

```text
http://localhost:8000/
http://localhost:8000/modules/est-prep/
```

Run the lightweight syntax and reference checks:

```bash
npm run check
```

The check validates JavaScript syntax, JSON syntax, EST Prep script/style links, and EST Prep asset references.
