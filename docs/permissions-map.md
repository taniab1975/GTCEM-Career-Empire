# Permissions Map

## Public User

Can:

- view welcome screen
- choose teacher or student login path

Cannot:

- access dashboards
- create student accounts
- access class data

## Teacher

Can:

- register with approved school email
- log in
- create and edit classes
- create student accounts
- reset student passwords
- assign or disable modules
- view teacher dashboard
- view class analytics
- view student progress and evidence

Cannot:

- see classes owned by other teachers unless later granted an admin role
- create teacher accounts for unapproved domains

## Student

Can:

- log in with username and password
- view own dashboard
- view own stats and assets
- play assigned modules
- view class leaderboard and community/global summaries if permitted

Cannot:

- self-register
- change class membership
- view private records of other students
- access teacher dashboards

## Future Optional Role: School Admin

Possible later role for:

- school-wide analytics
- teacher management
- cross-class oversight

Not required for first implementation.
